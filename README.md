# PinedIn Releases

Public release artifacts for [PinedIn](https://github.com/Zarl-prog/PinedIn) — the unignorable task OS.

This repo contains only built installer files. No source code.

## Downloads

Download the latest version from the [Releases](https://github.com/Zarl-prog/PinedIn-Releases/releases) page.

## Platforms

- **Windows:** `.msi` installer
- **macOS:** `.dmg` disk image
- **Linux:** `.AppImage` (universal) + `.deb` (Debian/Ubuntu)
