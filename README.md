## ✨ Features

| Feature | Description |
|---------|-------------|
| **Floating Cards** | Tasks appear as small always-on-top windows. Drag them anywhere. |
| **Urgency System** | Critical tasks shake and flash until done. Low priority sits quietly. |
| **Quick Add** | `Ctrl+Shift+Space` (Windows/Linux) or `Cmd+Shift+Space` (macOS) from anywhere. |
| **Daily Digest** | Morning briefing showing overdue, due today, and unfinished tasks. |
| **Pre-Schedule** | Schedule a task to appear at a future date and time. |
| **Zen Mode** | One click hides all non-task UI — just your tasks remain. |
| **Align** | Snap all floating cards into a clean grid. |
| **Workspaces** | Separate task lists for work, personal, side projects. |
| **Theme System** | Dark mode, Light mode, or follow system preference. |
| **System Tray** | Minimizes to tray. Always running, never in the way. |
| **Auto-Updater** | Ships with built-in updates via GitHub Releases. |
